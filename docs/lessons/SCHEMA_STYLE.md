# Schémas visuels — Les six cas

> Story 3.3 — Langage visuel unifié pour le parcours `les-six-cas`

Tous les SVG des leçons suivent ces conventions (alignées sur la leçon pilote).

| Élément | Valeur |
|---------|--------|
| `viewBox` | `0 0 480 280` (arbre) ou `0 0 480 160` (comparaison) |
| Trait / flèche | `#4F46E5`, `stroke-width="1.5"` |
| Boîte centrale | fill `#EEF0FF`, stroke `#4F46E5`, `rx="8"` |
| Boîte forme | fill `#FFFFFF`, stroke `#E8E4DC`, `rx="8"` |
| Labels rôle | `#5A5A5A`, `font-size="12"`, `font-weight="600"` |
| Mot russe | `font-family="serif"`, `font-size="15"` |
| Sujet (nominatif) | `#3B82F6` |
| Objet (accusatif) | `#EF7C5A` |
| Lieu / direction | `#22C55E` |
| Possession (génitif) | `#A78BFA` |
| Destinataire (datif) | `#F59E0B` |

**Logique** : un mot-racine en haut → branches vers les terminaisons selon le rôle. Les schémas comparent, ils ne listent pas.
